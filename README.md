A simple full-stack social media application where users can share updates, engage with content, and connect with others.

## Tech Stack
*   **Frontend**: React, Vite, Bootstrap
*   **Backend**: Node.js, Express
*   **Database**: MongoDB
*   **Authentication**: JWT (JSON Web Tokens)

## Key Features
*   **User Auth**: Secure Signup & Login
*   **Posts**: Create posts with text and images
*   **Interactions**: Like and Comment on posts in real-time
*   **Feed**: View latest posts from all users

## Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/Sumitchahar317/socialApp.git
cd socialApp
```

**2. Setup Backend**
```bash
cd server
npm install
npm start
```

**3. Setup Frontend**
```bash
cd frontend
npm install
npm run dev
```
